# Traffic Light Controller

## How It Works

- RED → Wait 3 seconds
- YELLOW → Wait 2 seconds
- GREEN → Wait 3 seconds
- Then prints "Traffic light cycle complete!"

---

## Python Code

```python
import time

print("RED - STOP")
time.sleep(3)

print("YELLOW - SLOW DOWN")
time.sleep(2)

print("GREEN - GO")
time.sleep(3)

print("Traffic light cycle complete!")
```
